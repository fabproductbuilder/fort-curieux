# Fort Curieux

Fort Curieux est une web app en préparation autour de deux piliers : créer et suivre son planning sportif, puis entretenir sa culture avec des sessions courtes sur de grands repères.

## Stack

- Next.js App Router
- TypeScript strict
- Tailwind CSS
- OpenNext pour Cloudflare Workers
- npm

## Commandes

```bash
npm run dev
```

```bash
npm run typecheck
```

```bash
npm run lint
```

```bash
npm run build
```

```bash
npm run preview
```

`npm run preview` construit l’application avec OpenNext puis lance une prévisualisation locale sur le runtime Cloudflare Workers.
